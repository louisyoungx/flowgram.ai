# Function: wrap

[Angle](/auto-docs/utils/modules/Angle.md).wrap

**wrap**(`angle`): `number`

Wrap an angle.

Wraps the angle to a value in the range of -PI to PI.

#### Parameters

| Name | Type | Description |
| :------ | :------ | :------ |
| `angle` | `number` | The angle to wrap, in radians. |

#### Returns

`number`

The wrapped angle, in radians.
